# ST Bank
Telegram-бот с механикой трейдинга в реальном времени

### Стек
| Задача | Технология |
|--|--|
| API для работы с TG | aiogram |
| База данных | PostgreSQL |
| Развёртывание | Docker |
| Работа с сервером | Flask |
| Типизация | pydantic |
