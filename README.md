## Economic Telegram bot with stock system

### Stack:

| **Task** | **Name** | **Version** |
| --- | --- | --- |
| Telegram Bot Library | aiogram | 3.6.0 |
| Database | [Supabase](supabase.com) | None |
| Integration with Google Sheets | google_api_python_client + oauth2client | 2.130.0 + 4.1.3 |
| Humanize numbers (1.234.412 → 1.2M) | humanize | 4.9.0 |
| Hosting | [Render](render.com) + Flask | None + 3.0.3 |

### Description:
#### Rus Vers
Бот предлагает получить опыт настоящего инвестора в валюты. Всего их 3: Российские Рубли (RUB), V и ST. Также в боте есть система открытия ящиков, где могут выпасть разные предметы. В этой системе реализована и система "повторок", которая даёт награду за повторное выпадение предмета.
В системе открытия боксов могут присутствовать немного странные (мемные) названия.


#### Eng Vers
The bot offers the experience of a real investor in currencies. There are 3 currencies in total: Russian Rubles (RUB), V and ST. The bot also has a system of opening boxes, where different items can fall out. In this system is implemented and the system of “repeats”, which gives a reward for the repeated falling out of the item.
The box opening system can have some slightly odd (meme) names.
